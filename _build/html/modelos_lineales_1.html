1. El Modelo Matem√°tico#

Fundamentos

La regresi√≥n lineal simple es un m√©todo estad√≠stico que modeliza la relaci√≥n entre una variable dependiente (y) y una variable independiente (x). Es ‚Äúsimple‚Äù porque solo hay una variable independiente y ‚Äúlineal‚Äù porque el modelo es lineal en los par√°metros.

1.1 Definici√≥n Formal#

El modelo se expresa como:

()#\[y_i = \beta_0 + \beta_1 x_i + \epsilon_i, \quad i = 1, 2, ..., n\]

donde:

  • \(y_i\) es la variable dependiente (respuesta)

  • \(x_i\) es la variable independiente (predictora)

  • \(\beta_0\) es el intercepto

  • \(\beta_1\) es la pendiente

  • \(\epsilon_i\) es el t√©rmino de error aleatorio

1.2 Supuestos del Modelo#

Los supuestos fundamentales son:

Supuestos Estad√≠sticos#

Supuesto

Expresi√≥n Matem√°tica

Interpretaci√≥n

1

\(E(\epsilon_i) = 0\)

El modelo es correcto y \(E(y_i) = \beta_0 + \beta_1 x_i\)

2

\(var(\epsilon_i) = \sigma^2\)

Homocedasticidad (varianza constante)

3

\(cov(\epsilon_i, \epsilon_j) = 0\)

Errores no correlacionados

Veamos un ejemplo con datos simulados:

# Generamos datos que cumplen los supuestos
np.random.seed(42)
n = 50
x = np.linspace(0, 10, n)
epsilon = np.random.normal(0, 1.5, n)  # Errores normales independientes
beta0_true, beta1_true = 2, 3

# Visualizaci√≥n

plt.ylabel('Variable dependiente (y)')

plt.legend()
plt.grid(True)

2. Estimaci√≥n de Par√°metros#

2.1 M√©todo de M√≠nimos Cuadrados#

Principio de M√≠nimos Cuadrados

El m√©todo busca los valores de \(\beta_0\) y \(\beta_1\) que minimizan la suma de cuadrados de los residuos:

\(\sum_{i=1}^n (y_i - \hat{y}i)^2 = \sum{i=1}^n (y_i - \hat{\beta}_0 - \hat{\beta}_1 x_i)^2\)

Objetivo

Encontrar los valores de \(\beta_0\) y \(\beta_1\) que minimicen la suma de los cuadrados de las diferencias entre los valores observados y los valores predichos.

Definimos la funci√≥n objetivo:

()#\[S(\beta_0, \beta_1) = \sum_{i=1}^n (y_i - \hat{y}_i)^2 = \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2\]

2.2 Condiciones de Primer Orden#

Para minimizar \(S(\beta_0, \beta_1)\), necesitamos que:

()#\[\begin{split}\begin{align*} \frac{\partial S}{\partial \beta_0} &= 0 \\ \frac{\partial S}{\partial \beta_1} &= 0 \end{align*}\end{split}\]

2.2.1 Primera Derivada Parcial#

Desarrollamos \(\frac{\partial S}{\partial \beta_0}\):

()#\[\begin{split}\begin{align*} \frac{\partial S}{\partial \beta_0} &= \frac{\partial}{\partial \beta_0} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2 \\ &= \sum_{i=1}^n \frac{\partial}{\partial \beta_0} (y_i - \beta_0 - \beta_1 x_i)^2 \\ &= \sum_{i=1}^n 2(y_i - \beta_0 - \beta_1 x_i)(-1) \\ &= -2\sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i) = 0 \end{align*}\end{split}\]

2.2.2 Segunda Derivada Parcial#

Desarrollamos \(\frac{\partial S}{\partial \beta_1}\):

()#\[\begin{split}\begin{align*} \frac{\partial S}{\partial \beta_1} &= \frac{\partial}{\partial \beta_1} \sum_{i=1}^n (y_i - \beta_0 - \beta_1 x_i)^2 \\ &= \sum_{i=1}^n \frac{\partial}{\partial \beta_1} (y_i - \beta_0 - \beta_1 x_i)^2 \\ &= \sum_{i=1}^n 2(y_i - \beta_0 - \beta_1 x_i)(-x_i) \\ &= -2\sum_{i=1}^n x_i(y_i - \beta_0 - \beta_1 x_i) = 0 \end{align*}\end{split}\]

2.3 Sistema de Ecuaciones Normales#

De ():

()#\[\sum_{i=1}^n y_i - n\beta_0 - \beta_1 \sum_{i=1}^n x_i = 0\]

De ():

()#\[\sum_{i=1}^n x_iy_i - \beta_0 \sum_{i=1}^n x_i - \beta_1 \sum_{i=1}^n x_i^2 = 0\]

2.4 Soluci√≥n del Sistema#

2.4.1 Obtenci√≥n de \(\beta_0\)#

De ():

()#\[\beta_0 = \frac{\sum_{i=1}^n y_i}{n} - \beta_1 \frac{\sum_{i=1}^n x_i}{n} = \bar{y} - \beta_1 \bar{x}\]

2.4.2 Obtenci√≥n de \(\beta_1\)#

Sustituyendo () en ():

()#\[\begin{split}\begin{align*} \sum_{i=1}^n x_iy_i - (\bar{y} - \beta_1 \bar{x})\sum_{i=1}^n x_i - \beta_1 \sum_{i=1}^n x_i^2 &= 0 \\ \sum_{i=1}^n x_iy_i - \bar{y}\sum_{i=1}^n x_i + \beta_1 \bar{x}\sum_{i=1}^n x_i - \beta_1 \sum_{i=1}^n x_i^2 &= 0 \end{align*}\end{split}\]

Reordenando t√©rminos:

()#\[\beta_1 = \frac{\sum_{i=1}^n x_iy_i - \bar{y}\sum_{i=1}^n x_i}{\sum_{i=1}^n x_i^2 - \bar{x}\sum_{i=1}^n x_i}\]

2.4.3 Forma Alternativa de \(\beta_1\)#

Podemos reescribir \(\beta_1\) en t√©rminos de desviaciones respecto a la media:

()#\[\begin{split}\begin{align*} \beta_1 &= \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2} \\ &= \frac{Cov(x,y)}{Var(x)} \end{align*}\end{split}\]

2.5 Verificaci√≥n de M√≠nimo#

Para confirmar que hemos encontrado un m√≠nimo y no un m√°ximo, verificamos las segundas derivadas:

()#\[\begin{split}\begin{align*} \frac{\partial^2 S}{\partial \beta_0^2} &= 2n > 0 \\ \frac{\partial^2 S}{\partial \beta_1^2} &= 2\sum_{i=1}^n x_i^2 > 0 \\ \frac{\partial^2 S}{\partial \beta_0\partial \beta_1} &= 2\sum_{i=1}^n x_i \end{align*}\end{split}\]

La matriz Hessiana es:

()#\[\begin{split}H = 2\begin{bmatrix} n & \sum x_i \\ \sum x_i & \sum x_i^2 \end{bmatrix}\end{split}\]

Es definida positiva, confirmando que hemos encontrado un m√≠nimo global.

2.6 Propiedades Algebraicas#

Las soluciones tienen las siguientes propiedades:

  1. \(\sum_{i=1}^n (y_i - \hat{y}_i) = 0\)

  2. \(\sum_{i=1}^n x_i(y_i - \hat{y}_i) = 0\)

  3. \(\sum_{i=1}^n \hat{y}_i = \sum_{i=1}^n y_i\)

  4. \(\hat{y} = \bar{y}\)

Note

Estas propiedades son consecuencia directa de las condiciones de primer orden y son fundamentales para la inferencia estad√≠stica en regresi√≥n lineal.

Implementemos estas f√≥rmulas:

def calcular_estimadores_mc(x, y):
    """Calcula los estimadores de m√≠nimos cuadrados"""
    x_mean, y_mean = np.mean(x), np.mean(y)
    beta1 = np.sum((x - x_mean) * (y - y_mean)) / np.sum((x - x_mean)**2)
    beta0 = y_mean - beta1 * x_mean
    return beta0, beta1

beta0_est, beta1_est = calcular_estimadores_mc(x, y)
print(f'Estimadores de m√≠nimos cuadrados:')
print(f'Œ≤‚ÇÄ: {beta0_est:.4f} (verdadero: {beta0_true})')
print(f'Œ≤‚ÇÅ: {beta1_est:.4f} (verdadero: {beta1_true})')

# Visualizaci√≥n del ajuste
plt.scatter(x, y, alpha=0.5, label='Datos')
plt.plot(x, beta0_true + beta1_true * x, 'r--', label='Verdadera')
plt.plot(x, beta0_est + beta1_est * x, 'g-', label='Ajustada')
plt.xlabel('x')
plt.ylabel('y')
plt.title('Comparaci√≥n de L√≠neas Verdadera y Ajustada')
plt.legend()
plt.grid(True)
plt.show()

Los estimadores de m√≠nimos cuadrados son insesgados y eficientes bajo los supuestos del modelo.

3. Propiedades de los Estimadores#

Propiedades Te√≥ricas

Bajo los supuestos del modelo, los estimadores de m√≠nimos cuadrados son:

  1. Insesgados: \(E(\hat{\beta}_0) = \beta_0\) y \(E(\hat{\beta}_1) = \beta_1\)

  2. De m√≠nima varianza entre todos los estimadores lineales insesgados

Las varianzas de los estimadores son:

()#\[\begin{split}\begin{align*} var(\hat{\beta}_1) &= \frac{\sigma^2}{\sum_{i=1}^n (x_i - \bar{x})^2} \\ var(\hat{\beta}_0) &= \sigma^2(\frac{1}{n} + \frac{\bar{x}^2}{\sum_{i=1}^n (x_i - \bar{x})^2}) \end{align*}\end{split}\]

3.1 Insesgamiento#

3.1.1 Insesgamiento de \(\hat{\beta}_1\)#

Teorema

El estimador \(\hat{\beta}_1\) es insesgado, es decir, \(E(\hat{\beta}_1) = \beta_1\)

Demostraci√≥n:

()#\[\begin{split}\begin{align*} E(\hat{\beta}_1) &= E\left[\frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}\right] \\ &= \frac{1}{\sum_{i=1}^n (x_i - \bar{x})^2}E\left[\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})\right] \end{align*}\end{split}\]

Dado que \(y_i = \beta_0 + \beta_1x_i + \epsilon_i\):

()#\[\begin{split}\begin{align*} E(y_i) &= \beta_0 + \beta_1x_i \\ E(\bar{y}) &= \beta_0 + \beta_1\bar{x} \end{align*}\end{split}\]

Sustituyendo:

()#\[\begin{split}\begin{align*} E(\hat{\beta}_1) &= \frac{1}{\sum_{i=1}^n (x_i - \bar{x})^2}\sum_{i=1}^n (x_i - \bar{x})(\beta_0 + \beta_1x_i - \beta_0 - \beta_1\bar{x}) \\ &= \frac{1}{\sum_{i=1}^n (x_i - \bar{x})^2}\sum_{i=1}^n (x_i - \bar{x})\beta_1(x_i - \bar{x}) \\ &= \beta_1\frac{\sum_{i=1}^n (x_i - \bar{x})^2}{\sum_{i=1}^n (x_i - \bar{x})^2} = \beta_1 \end{align*}\end{split}\]

3.1.2 Insesgamiento de \(\hat{\beta}_0\)#

Teorema

El estimador \(\hat{\beta}_0\) es insesgado, es decir, \(E(\hat{\beta}_0) = \beta_0\)

Demostraci√≥n:

()#\[\begin{split}\begin{align*} E(\hat{\beta}_0) &= E(\bar{y} - \hat{\beta}_1\bar{x}) \\ &= E(\bar{y}) - \bar{x}E(\hat{\beta}_1) \\ &= (\beta_0 + \beta_1\bar{x}) - \bar{x}\beta_1 \\ &= \beta_0 \end{align*}\end{split}\]

3.2 Varianzas de los Estimadores#

3.2.1 Varianza de \(\hat{\beta}_1\)#

()#\[\begin{split}\begin{align*} var(\hat{\beta}_1) &= var\left[\frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}\right] \\ &= \frac{1}{[\sum_{i=1}^n (x_i - \bar{x})^2]^2}var\left[\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})\right] \end{align*}\end{split}\]

Dado que \(var(y_i) = \sigma^2\) y los errores son independientes:

()#\[var(\hat{\beta}_1) = \frac{\sigma^2}{\sum_{i=1}^n (x_i - \bar{x})^2}\]

3.2.2 Varianza de \(\hat{\beta}_0\)#

()#\[\begin{split}\begin{align*} var(\hat{\beta}_0) &= var(\bar{y} - \hat{\beta}_1\bar{x}) \\ &= var(\bar{y}) + \bar{x}^2var(\hat{\beta}_1) - 2\bar{x}cov(\bar{y},\hat{\beta}_1) \end{align*}\end{split}\]

Despu√©s de desarrollar los t√©rminos:

()#\[var(\hat{\beta}_0) = \sigma^2(\frac{1}{n} + \frac{\bar{x}^2}{\sum_{i=1}^n (x_i - \bar{x})^2})\]

3.3 Teorema de Gauss-Markov#

Teorema de Gauss-Markov

Entre todos los estimadores lineales e insesgados de \(\beta_0\) y \(\beta_1\), los estimadores de m√≠nimos cuadrados tienen varianza m√≠nima.

3.3.1 Demostraci√≥n del Teorema#

Sea \(\tilde{\beta}_1\) otro estimador lineal insesgado de \(\beta_1\):

()#\[\begin{split}\begin{align*} \tilde{\beta}_1 &= \sum_{i=1}^n a_iy_i \\ E(\tilde{\beta}_1) &= \beta_1 \implies \sum_{i=1}^n a_ix_i = 1 \text{ y } \sum_{i=1}^n a_i = 0 \end{align*}\end{split}\]

Entonces:

()#\[var(\tilde{\beta}_1) = \sigma^2\sum_{i=1}^n a_i^2 \geq \frac{\sigma^2}{\sum_{i=1}^n (x_i - \bar{x})^2} = var(\hat{\beta}_1)\]

3.4 Propiedades de los Residuos#

Los residuos \(\hat{\epsilon}_i = y_i - \hat{y}_i\) tienen las siguientes propiedades:

  1. \(\sum_{i=1}^n \hat{\epsilon}_i = 0\)

  2. \(\sum_{i=1}^n x_i\hat{\epsilon}_i = 0\)

  3. \(\sum_{i=1}^n \hat{y}_i\hat{\epsilon}_i = 0\)

  4. \(Cov(\hat{y}_i, \hat{\epsilon}_i) = 0\)

Note

Estas propiedades son fundamentales para la inferencia estad√≠stica y la validaci√≥n del modelo.

3.5 Consecuencias Pr√°cticas#

  1. Eficiencia: Los estimadores MCO son los mejores estimadores lineales insesgados (BLUE).

  2. Precisi√≥n:

    • La precisi√≥n de \(\hat{\beta}_1\) aumenta con:

      • Mayor variabilidad en x (\(\sum_{i=1}^n (x_i - \bar{x})^2\) grande)

      • Menor varianza del error (\(\sigma^2\) peque√±o)

      • Mayor tama√±o muestral (n)

  3. Intervalo de Confianza:

    • Los intervalos de confianza ser√°n m√°s estrechos cuando:

      • Las varianzas sean menores

      • El tama√±o muestral sea mayor

4. Inferencia Estad√≠stica en Regresi√≥n Lineal#

4.1 Estimaci√≥n de la Varianza (œÉ¬≤)#

La varianza del error (œÉ¬≤) es un par√°metro crucial que mide la dispersi√≥n de los datos alrededor de la l√≠nea de regresi√≥n. Su estimaci√≥n es fundamental para:

  • Calcular errores est√°ndar de los coeficientes

  • Construir intervalos de confianza

  • Realizar pruebas de hip√≥tesis

import numpy as np
from scipy import stats

def estimate_variance(y, y_pred, n_params=2):
    """
    Estima la varianza del error en regresi√≥n lineal.
    
    Par√°metros:
    -----------
    y : array-like
        Valores observados
    y_pred : array-like
        Valores predichos
    n_params : int
        N√∫mero de par√°metros en el modelo (default=2 para regresi√≥n simple)
        
    Retorna:
    --------
    float
        Estimaci√≥n de œÉ¬≤
    """
    n = len(y)
    residuals = y - y_pred
    sigma2 = np.sum(residuals**2) / (n - n_params)
    return sigma2

# Ejemplo de uso
y_pred = beta0_est + beta1_est * x
sigma2_est = estimate_variance(y, y_pred)

4.2 Pruebas de Hip√≥tesis#

Las pruebas de hip√≥tesis nos permiten evaluar la significancia estad√≠stica de los coeficientes.

def test_beta1(beta1, x, sigma2, alpha=0.05):
    """
    Realiza prueba de hip√≥tesis para Œ≤‚ÇÅ
    
    H‚ÇÄ: Œ≤‚ÇÅ = 0 
    H‚ÇÅ: Œ≤‚ÇÅ ‚â† 0
    
    Par√°metros:
    -----------
    beta1 : float
        Estimaci√≥n de Œ≤‚ÇÅ
    x : array-like
        Variable independiente
    sigma2 : float
        Varianza estimada
    alpha : float
        Nivel de significancia
        
    Retorna:
    --------
    dict
        Resultados del test
    """
    n = len(x)
    # Error est√°ndar
    se = np.sqrt(sigma2 / np.sum((x - np.mean(x))**2))
    
    # Estad√≠stico t
    t_stat = beta1 / se
    
    # Grados de libertad
    df = n - 2
    
    # Valor p
    p_value = 2 * (1 - stats.t.cdf(abs(t_stat), df=df))
    
    # Intervalo de confianza
    t_crit = stats.t.ppf(1-alpha/2, df=df)
    ci = (beta1 - t_crit * se, beta1 + t_crit * se)
    
    return {
        't_statistic': t_stat,
        'p_value': p_value,
        'confidence_interval': ci
    }

# Ejemplo de uso
results = test_beta1(beta1_est, x, sigma2_est)
print(f"Estad√≠stico t: {results['t_statistic']:.4f}")
print(f"Valor p: {results['p_value']:.4e}")
print(f"IC 95%: [{results['confidence_interval'][0]:.4f}, {results['confidence_interval'][1]:.4f}]")

4.3 Intervalos de Predicci√≥n#

Los intervalos de predicci√≥n proporcionan un rango para futuras observaciones.

def prediction_interval(x_new, x, beta0, beta1, sigma2, alpha=0.05):
    """
    Calcula intervalo de predicci√≥n para nuevas observaciones
    
    Par√°metros:
    -----------
    x_new : float o array-like
        Nuevos valores de x para predicci√≥n
    x : array-like
        Valores originales de x
    beta0, beta1 : float
        Coeficientes estimados
    sigma2 : float
        Varianza estimada
    alpha : float
        Nivel de significancia
    
    Retorna:
    --------
    tuple
        (predicci√≥n, l√≠mite inferior, l√≠mite superior)
    """
    n = len(x)
    x_mean = np.mean(x)
    
    # Predicci√≥n puntual
    y_pred = beta0 + beta1 * x_new
    
    # Error est√°ndar de predicci√≥n
    se_pred = np.sqrt(sigma2 * (1 + 1/n + 
                     (x_new - x_mean)**2 / np.sum((x - x_mean)**2)))
    
    # Valor cr√≠tico
    t_crit = stats.t.ppf(1-alpha/2, df=n-2)
    
    # Intervalos
    pi_lower = y_pred - t_crit * se_pred
    pi_upper = y_pred + t_crit * se_pred
    
    return y_pred, pi_lower, pi_upper

# Ejemplo de uso
x_new = np.linspace(min(x), max(x), 100)
predictions = prediction_interval(x_new, x, beta0_est, beta1_est, sigma2_est)

plt.figure(figsize=(10, 6))
plt.scatter(x, y, alpha=0.5, label='Datos observados')
plt.plot(x_new, predictions[0], 'r-', label='Regresi√≥n')
plt.fill_between(x_new, predictions[1], predictions[2], 
                 alpha=0.2, label='Intervalo de predicci√≥n 95%')
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Regresi√≥n con Intervalos de Predicci√≥n')
plt.legend()
plt.grid(True)
plt.show()

5. Bondad de Ajuste en Regresi√≥n Lineal#

5.1 Coeficiente de Determinaci√≥n (R¬≤)#

El R¬≤ es una medida fundamental que indica la proporci√≥n de la variabilidad en los datos que es explicada por el modelo de regresi√≥n.

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

def calculate_r2(y, y_pred):
    """
    Calcula el coeficiente de determinaci√≥n R¬≤.
    
    Par√°metros:
    -----------
    y : array-like
        Valores observados
    y_pred : array-like
        Valores predichos
        
    Retorna:
    --------
    float
        Valor de R¬≤
    dict
        Componentes del c√°lculo (SST, SSR, SSE)
    """
    # Suma total de cuadrados (SST)
    ss_tot = np.sum((y - np.mean(y))**2)
    
    # Suma de cuadrados de la regresi√≥n (SSR)
    ss_reg = np.sum((y_pred - np.mean(y))**2)
    
    # Suma de cuadrados del error (SSE)
    ss_res = np.sum((y - y_pred)**2)
    
    # C√°lculo de R¬≤
    r2 = 1 - (ss_res / ss_tot)
    # Alternativamente: r2 = ss_reg / ss_tot
    
    return r2, {
        'SST': ss_tot,
        'SSR': ss_reg,
        'SSE': ss_res
    }

def plot_goodness_of_fit(y, y_pred, r2):
    """
    Genera visualizaciones para evaluar la bondad de ajuste.
    
    Par√°metros:
    -----------
    y : array-like
        Valores observados
    y_pred : array-like
        Valores predichos
    r2 : float
        Coeficiente de determinaci√≥n
    """
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(15, 6))
    
    # Gr√°fico de valores observados vs predichos
    ax1.scatter(y, y_pred, alpha=0.5)
    ax1.plot([min(y), max(y)], [min(y), max(y)], 'r--', 
             label='L√≠nea de referencia')
    ax1.set_xlabel('Valores Observados')
    ax1.set_ylabel('Valores Predichos')
    ax1.set_title(f'Observado vs Predicho (R¬≤ = {r2:.4f})')
    ax1.grid(True)
    ax1.legend()
    
    # Gr√°fico Q-Q de residuos
    residuals = y - y_pred
    stats.probplot(residuals, dist="norm", plot=ax2)
    ax2.set_title('Gr√°fico Q-Q de Residuos')
    
    plt.tight_layout()
    plt.show()

# Ejemplo de uso
r2, components = calculate_r2(y, y_pred)
plot_goodness_of_fit(y, y_pred, r2)

print("\nComponentes de la variabilidad:")
print(f"Variabilidad total (SST): {components['SST']:.4f}")
print(f"Variabilidad explicada (SSR): {components['SSR']:.4f}")
print(f"Variabilidad residual (SSE): {components['SSE']:.4f}")
print(f"\nR¬≤ = {r2:.4f}")

5.2 R¬≤ Ajustado#

El R¬≤ ajustado penaliza la adici√≥n de variables predictoras que no mejoran significativamente el modelo.

def adjusted_r2(r2, n, p):
    """
    Calcula el R¬≤ ajustado.
    
    Par√°metros:
    -----------
    r2 : float
        R¬≤ original
    n : int
        N√∫mero de observaciones
    p : int
        N√∫mero de predictores (sin incluir intercepto)
        
    Retorna:
    --------
    float
        R¬≤ ajustado
    """
    adj_r2 = 1 - (1 - r2) * ((n - 1) / (n - p - 1))
    return adj_r2

# Ejemplo de uso
n = len(y)  # n√∫mero de observaciones
p = 1       # n√∫mero de predictores (sin intercepto)
adj_r2 = adjusted_r2(r2, n, p)
print(f"R¬≤ ajustado: {adj_r2:.4f}")

5.3 Medidas Adicionales de Ajuste#

def additional_metrics(y, y_pred):
    """
    Calcula m√©tricas adicionales de bondad de ajuste.
    
    Par√°metros:
    -----------
    y : array-like
        Valores observados
    y_pred : array-like
        Valores predichos
        
    Retorna:
    --------
    dict
        Diccionario con diferentes m√©tricas
    """
    n = len(y)
    residuals = y - y_pred
    
    # Error cuadr√°tico medio (MSE)
    mse = np.mean(residuals**2)
    
    # Ra√≠z del error cuadr√°tico medio (RMSE)
    rmse = np.sqrt(mse)
    
    # Error absoluto medio (MAE)
    mae = np.mean(np.abs(residuals))
    
    # Error porcentual absoluto medio (MAPE)
    mape = np.mean(np.abs(residuals / y)) * 100
    
    return {
        'MSE': mse,
        'RMSE': rmse,
        'MAE': mae,
        'MAPE': mape
    }

# Ejemplo de uso
metrics = additional_metrics(y, y_pred)
for metric, value in metrics.items():
    print(f"{metric}: {value:.4f}")
